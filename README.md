# stock_trader
Реализация демо биржи на Vue JS

# Используемые технологии:

1. Vue js
  * vuex
    * store
    * actions
    * getters
    * mutations
  * vue-router
  * vue-components
  * vue-filters
  * transitions
  * axios
2. Bootstrap 3.3.7
3. Webpack
4. Babel

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
